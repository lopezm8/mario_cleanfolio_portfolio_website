Training a CNN to Detect Epileptic Spikes in EEG

Worked with a team of engineers to design a convolutional neural network (CNN) that detects and classifies epileptic spikes in EEG data. The data, sourced from the Temple University Events Corpus, underwent preprocessing steps like filtering, normalization, and segmentation. With the use of Python and PyTorch, the CNN model was trained and achieved a 60% accuracy in classifying waveforms. A complementary EEG plotting script was also developed to allow users to visualize and adjust waveforms. Throughout the project, comprehensive testing ensured model reliability, and clear documentation was provided for users.

Anesthesia Encylopedia Application

The Anesthesia_App is a tailored application website, crafted to provide multifaceted options for healthcare providers specializing in anesthesia. A pivotal component is its innovative web scraping microservice, which utilizes Axios and Cheerio modules to adeptly extract specific content from targeted web pages, enhancing user accessibility to pertinent information without direct navigation to secondary sites. Through a methodical approach, the microservice engages with website URLs, parses requisite HTML data, and then Cheerio extrapolates the desired text, streamlining the return of detailed descriptions (such as those from Oxford Medical Education) directly to the user on AnesthesiaApp. A meticulous UML Sequence Diagram and well-articulated code snippets provide a clear pathway for developers seeking to harness similar web scraping capabilities, ensuring usability and functionality within their applications.
